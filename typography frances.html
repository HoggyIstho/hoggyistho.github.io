Tools we use

Html 5

Quand on parle de HTML5, la première chose qu'il faut savoir est qu'il s'agit de la dernière version de la technologie HTML, dont les initiales correspondent à « HyperText Markup Language », qui a la signification suivante:

  • HyperTexte, dont le sens est hypertexte, qui n'est rien de plus qu'un texte qui renvoie à un autre contenu, qui peut être un autre texte ou un autre fichier. C'est la base du fonctionnement du web tel que nous le connaissons, qui n'est rien de plus que des pages et des ressources interconnectées.
  • Le balisage, ce qui signifie marquer ou baliser, puisque toutes les pages Web sont construites sur la base de balises, des premières versions aux dernières balises HTML5. Un exemple de balise HTML est celui qui identifie un paragraphe, qui est composé de la balise, du contenu de la balise et de la fermeture du paragraphe.
  • Langage, dont le sens est langage, car HTML est un langage, c'est-à-dire qu'il a ses règles, sa structure et une série de conventions qui nous aident à définir à la fois la structure et le contenu d'un site Web.

Quelque chose d'important à garder à l'esprit et avec lequel il ne faut pas être confondu, c'est que parce que HTML est un langage, cela ne signifie pas qu'il s'agit d'un langage de programmation. HTML ne l'est pas, car il n'a pas de structures de langage de programmation telles que des boucles, des conditions, des fonctions, etc.

Css

CSS (Cascading Style Sheets) est ce qu'on appelle un langage de feuille de style en cascade et est utilisé pour styliser des éléments écrits dans un langage de balisage tel que HTML. CSS sépare le contenu de la représentation visuelle du site.

CSS a été développé par le W3C (World Wide Web Consortium) en 1996 pour une raison très simple. HTML n'a pas été conçu pour avoir des balises pour aider à formater la page. Il est fait juste pour écrire le balisage du site.

Des balises telles que la police ont été incluses dans la version HTML 3.2, ce qui a causé beaucoup de problèmes aux développeurs. Étant donné que les sites Web avaient des polices, des arrière-plans de couleur et des styles différents, le processus de réécriture du code était long, pénible et coûteux. Par conséquent, CSS a été créé par le W3C pour résoudre ce problème.

La relation entre HTML et CSS est très forte. Étant donné que HTML est un langage de balisage (c'est-à-dire qu'il constitue la base d'un site) et que CSS met l'accent sur le style (toute la partie esthétique d'un site Web), ils vont de pair.

CSS n'est techniquement pas une nécessité, mais vous ne voulez pas avoir un site uniquement HTML car il aurait l'air complètement nu.

JavaScript

Javascript est un langage qui peut être utilisé par les professionnels et ceux qui débutent dans le développement et la conception de sites Web. Il ne nécessite pas de compilation puisque le langage fonctionne côté client, les navigateurs se chargent d'interpréter ces codes.

Beaucoup confondent Javascript avec Java, mais les deux langages sont différents et ont leurs propres caractéristiques. Javascript a l'avantage d'être intégré à n'importe quelle page web, il peut être exécuté sans avoir besoin d'installer un autre programme à afficher.

Java, pour sa part, a pour principale caractéristique d'être un langage indépendant de la plate-forme. Vous pouvez créer toutes sortes de programmes qui peuvent être exécutés sur n'importe quel ordinateur du marché : Linux, Windows, Apple, etc. En raison de ses caractéristiques, il est également largement utilisé pour Internet.

En résumé, on peut dire que Javascript est un langage interprété, basé sur des prototypes, tandis que Java est un langage plus orienté objet.

Bootstrap

Bootstrap est un framework frontal utilisé pour développer des applications Web et des sites mobiles first, c'est-à-dire avec une mise en page qui s'adapte à l'écran de l'appareil utilisé par l'utilisateur. Apprenez tout sur cette technologie et comment l'utiliser!

Bootstrap est un framework CSS utilisé dans les applications frontales - c'est-à-dire dans l'écran de l'interface utilisateur - pour développer des applications qui s'adaptent à n'importe quel appareil.

Dans WordPress, par exemple, il peut être installé en tant que thème ou utilisé pour le développement de plugins ou même en leur sein pour styliser leurs fonctions. Le framework a pour but d'offrir à l'utilisateur une expérience plus agréable lors de la navigation sur un site.

Pour cette raison, il dispose de plusieurs ressources pour configurer les styles des éléments de la page de manière simple et efficace, en plus de faciliter la construction de pages qui, à la fois, sont adaptées pour le Web et pour les appareils mobiles.

Ce qui précède montre pourquoi il est important de connaître une structure potentielle de ce type.

GitHub

GitHub est une plate-forme d'hébergement, détenue par Microsoft, qui offre aux développeurs la possibilité de créer des référentiels de code et de les enregistrer dans le cloud en toute sécurité, à l'aide d'un système de contrôle de version, appelé Git.

Facilite l'organisation des projets et permet la collaboration multi-développeurs en temps réel. Autrement dit, il nous permet de centraliser le contenu du référentiel afin de collaborer avec les autres membres de notre organisation.

GitHub est basé sur le système de contrôle de version distribué Git, on peut donc compter sur ses fonctions et ses outils, bien que GitHub offre plusieurs options supplémentaires et que son interface soit beaucoup plus simple à utiliser, il n'est donc pas absolument nécessaire que les personnes qui l'utilisent aient de bonnes connaissance technique.

Gardez un œil sur les nouveaux outils que nous utiliserons à l'avenir seront téléchargés.
