Инструменты, которые мы используем

Html 5

Когда мы говорим о HTML5, первое, что мы должны знать, это то, что это последняя версия технологии HTML, инициалы которой соответствуют «языку разметки гипертекста», что имеет следующее значение:

  • HyperText, whose meaning is hypertext, which is nothing more than a text that links to other content, which can be another text or another file. This is the basis of the operation of the web as we know it, which is nothing more than interconnected pages and resources.
  • Markup, which means mark or tag, since all web pages are built based on tags, from the first versions to the latest HTML5 tags. An example of an HTML tag is the one that identifies a paragraph, which is made up of the tag, the content of the tag, and the closing of the paragraph.
  • Language, whose meaning is language, because HTML is a language, that is, it has its rules, it has its structure and a series of conventions that help us define both the structure and the content of a website.

Something important to keep in mind and with which one should not be confused, is that because HTML is a language it does not mean that it is a programming language. HTML is not, as it does not have programming language structures such as loops, conditions, functions, and so on.

Css

CSS (каскадные таблицы стилей) - это так называемый язык каскадных таблиц стилей, который используется для стилизации элементов, написанных на языке разметки, таком как HTML. CSS отделяет контент от визуального представления сайта.

CSS был разработан консорциумом W3C (World Wide Web Consortium) в 1996 году по очень простой причине. HTML не предназначен для использования тегов для форматирования страницы. Сделано просто для написания разметки для сайта.

Такие теги, как font, были включены в HTML версии 3.2, и это вызвало массу проблем у разработчиков. Поскольку у веб-сайтов были разные шрифты, цветовой фон и стили, процесс переписывания кода был долгим, болезненным и дорогостоящим. Следовательно, CSS был создан W3C для решения этой проблемы.

Связь между HTML и CSS очень сильна. Поскольку HTML - это язык разметки (то есть он составляет основу сайта), а CSS подчеркивает стиль (всю эстетическую часть веб-сайта), они идут рука об руку.

Технически CSS не является необходимостью, но вы не хотите иметь сайт только в формате HTML, так как он будет выглядеть совершенно пустым.

JavaScript

Javascript - это язык, которым могут пользоваться профессионалы и новички в разработке и дизайне веб-сайтов. Он не требует компиляции, так как язык работает на стороне клиента, интерпретация этих кодов берут на себя браузеры.

Многие путают Javascript с Java, но оба языка разные и имеют свои уникальные особенности. Преимущество Javascript в том, что он встроен в любую веб-страницу, он может быть выполнен без необходимости установки другой программы для отображения.

Со своей стороны, Java имеет основную характеристику независимого от платформы языка. Вы можете создавать всевозможные программы, которые можно запускать на любом компьютере, представленном на рынке: Linux, Windows, Apple и т. Д. Благодаря своим характеристикам он также широко используется в Интернете.

Подводя итог, можно сказать, что Javascript - это интерпретируемый язык, основанный на прототипах, тогда как Java - более объектно-ориентированный язык.

Bootstrap

Bootstrap - это интерфейсный фреймворк, используемый для разработки веб-приложений и мобильных сайтов, то есть с макетом, который адаптируется к экрану устройства, используемого пользователем. Узнайте все об этой технологии и о том, как ее использовать!

Bootstrap - это CSS-фреймворк, используемый во интерфейсных приложениях, то есть на экране пользовательского интерфейса, для разработки приложений, которые адаптируются к любому устройству.

В WordPress, например, его можно установить как тему или использовать для разработки плагинов или даже внутри них для стилизации их функций. Цель фреймворка - предложить пользователю более приятные впечатления от просмотра сайта.

По этой причине в нем есть несколько ресурсов для простой и эффективной настройки стилей элементов страницы, а также для облегчения создания страниц, которые в то же время адаптированы для Интернета и для мобильных устройств.

Выше показано, почему важно знать о потенциальной структуре этого типа.

GitHub

GitHub - это хостинговая платформа, принадлежащая Microsoft, которая предлагает разработчикам возможность создавать репозитории кода и безопасно сохранять их в облаке с помощью системы контроля версий, называемой Git.

Облегчает организацию проекта и обеспечивает совместную работу нескольких разработчиков в режиме реального времени. Другими словами, это позволяет нам централизовать содержимое репозитория для сотрудничества с другими членами нашей организации.

GitHub основан на распределенной системе контроля версий Git, поэтому на его функции и инструменты можно положиться, хотя GitHub предлагает несколько дополнительных опций, а его интерфейс намного проще в использовании, поэтому не обязательно, чтобы у людей, которые его используют, были отличные технические знания.
